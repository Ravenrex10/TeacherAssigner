# 🧑‍🏫 Teacher Yard Dutty Assigner Desktop App

Welcome to **TeacherAssigner**, a desktop application that automates the process of assigning teachers to yard duty schedules! This project is built using **Electron.js**, making it cross-platform and easy to run on Windows, Mac, and Linux.
This is still WIP.

## 🚀 Features

- ⚡ **Automated Teacher Assignments**: Quickly assigns teachers randomly.
- 🗂️ **Manage Schedules**: Organize teacher schedules efficiently with no conflicts.
- 🖥️ **Cross-Platform**: Runs on Windows, macOS, and Linux thanks to Electron.js.
- 🎨 **User-Friendly Interface**: Simple, clean, and easy-to-use desktop application.

## 🛠️ Tech Stack

- 🟢 **Electron.js**: Cross-platform desktop app framework
- 🔥 **Node.js**: Backend logic
- 📄 **HTML, CSS, JavaScript**: For building the UI
- 💾 **SQLite**: Lightweight database for storing teacher and class information

## 📥 Installation

Follow these steps to set up the app locally:

1. **Clone the repo**:
    ```bash
    git clone https://github.com/Ravenrex10/TeacherAssigner
    ```
2. **Install dependencies**:
    ```bash
    cd TeacherAssigner
    npm install
    ```
3. **Run the app**:
    ```bash
    npm start
    ```

## 🧑‍💻 Usage

- Launch the app and input teacher information in the teachers tab.
- Click on the "Assign Teachers" button in the timetable tab, and the app will automatically assign teachers based on the data you provide.
-  Modify the schedule if needed.
- Review the schedule and export if needed.
